# 智能数据分析助手系统

> 生成时间: 2025-12-19 17:56:58
> 角色数量: 4

## 需求描述

我需要一个数据分析助手。

数据类型：销售数据
分析目的：业务洞察 / 决策支持 / 异常监控 / 趋势预测]
输出形式：分析报告 / 可视化图表 / 数据看板]

核心需求：
- 从数据中发现有价值的洞察
- 用清晰的方式呈现分析结果
- 提供可执行的业务建议

## 角色列表

| 序号 | 角色名称 | 类型 | 描述 |
|------|----------|------|------|
| 1 | 数据预处理师 | support | 负责数据清洗、验证、转换和初步统计，确保数据质量 |
| 2 | 核心分析师 | core | 负责深度数据分析，挖掘业务洞察，发现趋势和模式 |
| 3 | 异常监控师 | quality | 负责监控分析结果中的异常、风险和趋势突变，提供预警和原因分析 |
| 4 | 洞察顾问 | core | 负责将分析结果转化为业务洞察和可执行建议 |
